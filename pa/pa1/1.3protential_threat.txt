This is not an undefined behaviour.

According to ISO/IEC 9899:201x (p51, pdf p69), 6.3.1.3.2, "Otherwise, if the
new type is unsigned, the value is converted by repeatedly adding or
substracting one more than the maximum value that can be represented in the
new type until the value is in the range of the new type.".

我的中文理解为：“将原值（将原变量的比特序列按原类型，数学意义上的值）重复执
行（数学意义上的）加或减（新类型值域上界加1）这个值，直到这个值落在新变量的值
域中。”

因此，传入 -1 时，将这个值加 $2^64$ 得 $2^64 - 1$，即为 `UINT64_MAX`。
