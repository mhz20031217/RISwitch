一个程序从哪里开始执行呢?
答：如果是一个 C 程序，我们通常认为是从 `main` 开始的。但是，这是操作系统和 C 语言运行时环境提供的抽象。
尝试用 `objdump` 查看 hello 程序的符号表，可见以下条目：

```
    29: 0000000000001060    38 FUNC    GLOBAL DEFAULT   16 _start
```

在程序被装入到 `main` 函数被调用，进行了一些初始化和准备工作，`_start` 函数是由 C 库提供的入口函数。

在 x86-64-linux-gnu 平台上，使用 gcc 编译器，通过 `objdump` 可以观察到程序实际是从 `_start` 开始的；

C 程序真正的开始位置是 `_start` 函数；
